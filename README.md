> Tal Bamberger, Efrat Muller, Yadid M. Algavi, Ashlee Greenier, Christine Adjangba, Elizabeth Slikas, Layla Brassington et al. “Mapping the canine microbiome: Insights from the Dog Aging Project.” bioRxiv (2024): 2024-12.
